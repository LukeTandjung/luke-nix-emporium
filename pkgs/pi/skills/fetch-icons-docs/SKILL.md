---
name: fetch-icons-docs
description: Fetch HeroIcons or RadixIcons documentation when mapping icons from Figma designs to React components.
version: 0.1.0
---

When the user or project instructions require you to work with an external React icon library (like HeroIcons, RadixIcons), automatically:

1. Identify the official website for the icon library
2. Fetch the documentation

Examples:

During a Figma design conversion to code, an icon from an icon design file was used ->
1. Identify the name of the icon, and the library it comes from
2. Fetch the website of the library
3. Search for the icon name, and the corresponding React component of the icon to use.
